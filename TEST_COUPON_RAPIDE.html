5️⃣ Instructions

Avant de tester :
1. Assurez-vous que le backend est démarré (port 8080)
2. Assurez-vous que vous êtes connecté dans l'application
3. Exécutez le script SQL de nettoyage dans phpMyAdmin
4. Rafraîchissez cette page (F5)

Ordre des tests :
1. Vérifier le token (doit être présent)
2. Valider un coupon (doit retourner status 200)
3. Vérifier le panier (doit contenir des articles)
4. Créer une commande de test

En cas d'erreur :
- Copiez le message d'erreur complet
- Vérifiez les logs du backend dans le terminal
- Vérifiez que les coupons TEST1 et TEST2 existent dans la base
