  → Sauvegarde de la remise lors de la création de la commande (ligne 186)
  → Ajout de la remise dans la réponse (ligne 377)

  → Ajout du champ "remise" dans le DTO

🎨 FRONTEND (Angular)
─────────────────────────────────────────────────────────────────────────────

✓ checkout.component.ts
  → Correction du getter cartDeliveryFee (retourne 5.0 par défaut)
  → Correction du getter cartTotal (calcul avec frais de livraison)

💾 BASE DE DONNÉES (MySQL)
─────────────────────────────────────────────────────────────────────────────

✓ Script SQL : AJOUTER_COLONNE_REMISE.sql
  → Ajout de la colonne "remise" dans la table "orders"
  → Type : DECIMAL(19,2)
  → Valeur par défaut : 0.00
