## 📞 Support

### Questions Fréquentes

**Q : Dois-je tout changer d'un coup ?**  
R : Non ! Commencez par les composants prioritaires (Cart, Checkout). Le composant Products est déjà fait comme exemple.

**Q : Les variables CSS sont obligatoires ?**  
R : Oui, pour maintenir la cohérence. Utilisez toujours `var(--color-primary)` au lieu de `#4CAF50`.

**Q : BEM est-il obligatoire ?**  
R : Oui, pour la maintenabilité. Cela rend le code plus clair et évite les conflits de styles.

**Q : Puis-je modifier la charte ?**  
R : Oui, mais documentez les changements dans `CHARTE_GRAPHIQUE.md` pour que toute l'équipe soit au courant.
